# KiCad Automation Container
Container that contains KiCad and other tools for automation tasks.

## Included Tools
* kibot

## Supported Platforms
* linux/arm64
* linux/amd64

## Container Tags
* latest (Latest build)
* x.x.x (Latest build of specified KiCad version)
* x.x.x-{hash} ({KiCad version}-{Commit short hash})
